# passthrough

passthrough(mixed $value) :mixed

🇯🇵 🇺🇸
$valueをそのまま返します。

# パラメータ

# $value

返したい値。

# 例

>>> Faker\Factory::create('ja_JP')->passthrough('日本語')
=> "日本語"
>>> Faker\factory::create()->passthrough(['a' => 'A'])
=> [
     "a" => "A",
   ]
