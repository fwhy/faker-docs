# jobTitle

jobTitle() :string

🇯🇵 🇺🇸
職名を返します。

# パラメータ

なし

# 例

>>> Faker\Factory::create('ja_JP')->jobTitle
=> "alias"
>>> Faker\Factory::create()->jobTitle
=> "Numerical Tool Programmer OR Process Control Programmer"

# 注意

注意:
