# sha256
```php
sha256() :string
```
:jp: :us:  
SHA-256ハッシュを生成します。

## パラメータ
なし

## 例
```php
>>> Faker\Factory::create('ja_JP')->sha256()
=> "ea935fe630daf6340840ebbcb5d0886828b468f91bf3e542c289827cea1e7bef"
>>> Faker\Factory::create()->sha256()
=> "b59c84b99b109bed7201f2583381a85bbd9b2ece02d8a6725ec7830cee6aa450"
```
